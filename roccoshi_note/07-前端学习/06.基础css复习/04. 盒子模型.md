# 04. 盒子模型

## 1 | 边框

| 属性         | 作用       |
| ------------ | ---------- |
| border-width | 边框的粗细 |
| border-style | 边框的样式 |
| border-color | 边框的颜色 |

## 2 | 内外边距

外边距 : **margin** ( 上下, 左右 `margin: 0 auto`->实现居中 (上下0 左右auto) )

内边距 : **padding** 复合属性 ( 上, 右, 下, 左  -> 即顺时针 )

实现行内元素和行内块元素的水平居中 : 给父元素添加`text-align: center`
