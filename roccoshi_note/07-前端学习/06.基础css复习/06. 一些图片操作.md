# 06. 一些图片操作

## 1 | 常见的图片格式

| 图像格式    | 说明                                                         |
| ----------- | ------------------------------------------------------------ |
| jpg \| jpeg | 对色彩的信息保留好, 高清, 颜色较多 ( 产品类的图片 )          |
| gif         | 最多只能存储256色, 通常用来显示简单图形以及字体, 但是可以保存**透明背景和动画效果** |
| png         | 网络图形格式, 结合了gif和jpeg的优点, 具有存储形式丰富的特点, 能够保持**透明背景** |
| psd         | photoshop的专用格式, 里面可以存放图层, 通道, 遮罩等多种设计稿 |
