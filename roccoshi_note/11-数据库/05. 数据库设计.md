# 05. 数据库设计

## 1 | 数据库设计的流程

| 1-需求分析 | 2-概念结构设计 | 3-逻辑结构设计 | 4-物理结构设计 |
| ---------- | -------------- | -------------- | -------------- |
|            | ER图           | ER图->关系模型 |                |

## 2 | ER图

三种表示

|      | 表示方式 |
| ---- | -------- |
| 实体 | 矩形     |
| 属性 | 圆形     |
| 联系 | 菱形     |

三种不同的联系

- 一对一联系
- 一对多联系
- 多对多联系

## 3 | ER图向关系模型的转换

| 联系                   | 转换                                                         |
| ---------------------- | ------------------------------------------------------------ |
| 1 : 1                  | 两实体**任意一端添加另一端的主键**                           |
| 1 : N                  | 在**N端添加另一端的主键**                                    |
| N : M                  | 将**联系转化为实体**, 并在实体中加入联系两端实体的主键       |
| 1 : 1 : N \| N : M : P | 同 1 : N \| N : M (在N端添加另外两端主键 \| 联系转化为实体添加三端实体主键) |
